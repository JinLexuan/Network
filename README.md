# Network-Programming-in-CPP
This repo is for learning network programming in C++.

To download this repo:
``` bash
git clone https://github.com/JinLexuan/Network-Programming-in-CPP.git
```

Enter the folder
``` bash
cd Network-Programming-in-CPP
```

Compile
``` bash
mkdir build
cd build
cmake ..
make -j32
```
